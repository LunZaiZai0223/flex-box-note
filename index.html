FLEX-WRAP

display: flex 狀態下 flex-wrap 預設是 nowrap 。
- nowrap 的狀態：會強迫容器內第一層的子元素排成一行(超出範圍也不會換行)，所以 nowrap 會無視設定的 width 及 height。
- wrap 的狀態：會正常顯示容器內第一層子元素的 width 及 height ，超過範圍就會自動換行。

ALIGN-ITEMS （以 flex-direction 為 row 時的範例）

align-items 是屬於交錯軸的設定與主線軸不同方向。比方來說，當主線軸為水平，那麼 align-items 就是設定垂直的位置；當主軸線為垂直，那麼 align-items 則是設定水平的位置。

- stretch 會使容器內第一層子元素佔滿容器的空間(但如果元素本身有設定 width 及 height 則會保留設定)。
- baseline 會依照元素內的文字為基準對齊。

ALIGN-CONTENT （以 flex-direction 為 row 時的範例）

align-items 的多行版本，有「多行」才會執行。如果沒有 flex-wrap: wrap or wrap-reverse 就不會有「多行」的情況，所以就不會執行。

FLEX-GROW

flex-grow(default 0)
子元素的擴展性：如果容器還有空間的話會依照數字的比例擴展。如果全部的子元素 flex-grow:1 ，則每個子元素擴展的範圍則相同，如果有其中一個子元素設為 2 ，則該子元素擴展的範圍會最大。
- 負數的數字無效

FLEX-SHRINK

flex-shrink(default 0)
子元素的收縮性：如果容器空間不夠時，子元素會依照數字的比例縮小。如果全部的子元素 flex-shrink:1 ，則每個子元素縮小的範圍則相同，如果有其中一個子元素設為 2 ，則該子元素縮小的範圍會最大。
- 負數的數字無效
